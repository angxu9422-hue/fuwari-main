WSL2 一键安装

对于 Windows 10/11 用户，以管理员身份打开 PowerShell，执行以下命令即可自动安装 Ubuntu：

```bash

wsl --install

```

安装完成后在开始菜单启动 Ubuntu，设置用户名和密码即可。

十四、新手快速上手指南

如果你是初学者，面对这么多工具可能会感到困惑。以下是一个最小可行环境的搭建路径：

Windows 用户

1. 安装 WSL2（以管理员身份打开 PowerShell）：

   ```bash

   wsl --install

   ```

2. 安装 VS Code：下载地址

3. 在 WSL 中安装 Python：

   ```bash

   sudo apt update

   sudo apt install python3 python3-pip

   ```

4. 在 WSL 中安装 Node.js：

   ```bash

   curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -

   sudo apt install -y nodejs

   ```

5. 安装 VS Code WSL 插件：在 VS Code 扩展商店搜索 "WSL" 并安装

macOS 用户

1. 安装 Homebrew：

   ```bash

   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

   ```

2. 安装 VS Code：`brew install --cask visual-studio-code`

3. 安装 Python：`brew install python`

4. 安装 Node.js：`brew install node`

Linux 用户

1. 更新系统：`sudo apt update && sudo apt upgrade`（Debian/Ubuntu）

2. 安装 VS Code：从官网下载 .deb 包

3. 安装 Python：`sudo apt install python3 python3-pip`

4. 安装 Node.js：`sudo apt install nodejs npm`

验证安装

打开终端，执行以下命令检查环境是否正常：

```bash

# 检查 Python

python --version

pip --version

# 检查 Node.js

node --version

npm --version

# 检查 Git

git –version
